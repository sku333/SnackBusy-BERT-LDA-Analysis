# 硬折扣零食店消费者评论数据挖掘分析报告

**数据来源**：大众点评、美团（上海零食很忙、好想来门店）  
**有效样本**：1,375 条用户评论  
**分析方法**：传统NLP方法（Jieba + SnowNLP + LDA）vs 深度学习方法（RoBERTa + BERT + LDA）

---

## 一、情感分析结果对比

### 1.1 方法说明

| 方法 | 模型 | 原理 |
|------|------|------|
| 原方法 | SnowNLP | 基于朴素贝叶斯的规则模型，输出0-1情感分值 |
| 新方法 | RoBERTa | 在京东电商评论上微调的深度学习模型，直接输出positive/negative标签 + 置信度 |

### 1.2 结果对比

| 情感类别 | 原方法 (SnowNLP) | 新方法 (RoBERTa) | 差异分析 |
|---------|----------------|----------------|---------|
| 正向 | 73.8%（1,015条）| **89.8%**（1,235条）| +16个百分点 |
| 中性 | 3.9%（54条）| N/A（二分类模型）| 原中性评论被分配至正/负 |
| 负向 | 22.3%（306条）| **10.2%**（140条）| -12个百分点 |
| 平均置信度 | 无 | **96.93%** | 模型预测把握度极高 |

### 1.3 解读

RoBERTa识别出**更高比例的正向评论（89.8%）**，与SnowNLP结果（73.8%）存在显著差异。原因分析：

- SnowNLP基于通用语料训练，对电商口语化表达（"还不错""蛮便宜的""挺多选择"）的语义理解较弱，倾向将模糊表达判为负向
- RoBERTa在京东评论上微调，能准确识别电商领域的正面表达，包括隐含肯定的评论
- **真实情感倾向**：89.8%的正向比例更符合消费者主动评价行为（有正面体验才更愿意写评论）的规律

**数据挖掘结论**：硬折扣零食店整体口碑良好，超过九成评论为正向，消费者满意度较高。

## 二、主题挖掘结果对比

### 2.1 好评主题对比

#### 原论文方法（Jieba + LDA）

| 主题 | 核心关键词 | 主题标签 | 解读 |
|------|----------|---------|------|
| 主题1 | 价格、便宜、性价比、方便、不错 | **性价比与便利性** | 消费者最关注价格优势和购物便利程度 |
| 主题2 | 环境、服务、品种、实惠、整洁 | **门店体验** | 店内环境整洁、品种丰富是正面评价重点 |
| 主题3 | 好吃、口味、品类、丰富、味道 | **产品口味** | 产品本身的口感和品类丰富度获得认可 |

#### 新方法（Jieba + LDA，本次复现）

| 主题 | 核心关键词 | 主题标签 |
|------|----------|---------|
| 主题1 | 便宜、新开、喜欢、饮料、想来 | **价格吸引与复购意愿** |
| 主题2 | 口味、好吃、方便、干净、划算 | **综合体验满意** |
| 主题3 | 货架、实惠、店员、好吃、丰富 | **陈列与服务** |

**两次LDA结果高度一致**：价格/性价比、门店体验、产品口味是好评的三大驱动因素。

### 2.2 差评主题对比

#### 原论文方法（Jieba + LDA）

| 主题 | 核心关键词 | 主题标签 | 解读 |
|------|----------|---------|------|
| 主题1 | 收银、结账、排队、收银员、袋子 | **收银效率问题** | 排队结账慢、收银员服务是最集中的差评点 |
| 主题2 | 服务、态度、商品、品牌、营业员 | **员工服务态度** | 部分员工态度问题引发不满 |
| 主题3 | 会员、价格、便宜、货架、品种 | **会员权益与货架管理** | 会员制度不透明、商品补货不及时 |

#### 新方法（Jieba + LDA，本次复现）

| 主题 | 核心关键词 | 主题标签 |
|------|----------|---------|
| 主题1 | 态度、收银员、袋子、问题 | **收银服务态度** |
| 主题2 | 兑换、结果、巧克力 | **促销兑换问题** |
| 主题3 | 店员、袋子、核销、买单、结账 | **收银流程体验** |

#### BERT语义聚类新发现

BERT方法在差评中额外识别出一个**LDA未发现的细粒度主题**：

| 聚类 | 关键词 | 主题标签 | 新洞察 |
|------|--------|---------|-------|
| 聚类1 | 剪刀、撕开、果冻、勺子 | **包装开封困难** | 部分产品包装设计不友好，消费者需借助工具才能开封 |
| 聚类3 | 插队、当时、队伍、核销 | **排队秩序管理** | 高峰期收银队伍秩序混乱，存在插队现象 |

## 三、综合数据挖掘结论

### 3.1 消费者满意度驱动因素（好评）

```
优先级排序（基于主题词频与情感强度）：

第1位：价格优势与性价比 ── 便宜、实惠、划算是最高频正面词
第2位：产品品类丰富度 ── 零食种类多、品牌齐全是核心优势
第3位：门店环境整洁 ── 货架整齐、店内干净影响购物体验
第4位：产品口味 ── 好吃、口味满意直接驱动复购
```

### 3.2 消费者痛点（差评）

```
优先级排序（基于差评集中度）：

第1位：收银效率与等待体验 ── 排队结账是最集中的差评
第2位：员工服务态度 ── 部分员工态度冷漠、不主动
第3位：包装设计问题 ── 部分零食开封困难（BERT新发现）
第4位：会员权益不透明 ── 促销规则复杂、兑换流程混乱
```

### 3.3 业务改进建议

| 痛点 | 建议措施 | 优先级 |
|------|---------|-------|
| 收银排队慢 | 增设自助收银机；高峰期增派收银员；优化收银系统响应速度 | 高 |
| 员工服务态度 | 加强员工培训；建立服务评价机制；绩效与顾客评分挂钩 | 高 |
| 包装开封困难 | 在门店提供剪刀/开袋工具；向供应商反馈包装优化需求 | 中 |
| 会员权益复杂 | 简化积分兑换流程；培训员工熟悉会员系统；在货架标注会员价 | 中 |
| 货架补货 | 建立实时库存提醒；优化热门商品补货频率 | 低 |

## 四、正大杯问卷设计建议

基于数据挖掘发现的5大消费者关注维度，建议问卷从以下角度设计：

### 模块一：消费行为基本信息
> 目的：了解受访者画像与消费频次

- 您的年龄段（18-24 / 25-30 / 31-35 / 35岁以上）
- 您在上海硬折扣零食店的消费频次（每周多次 / 每周一次 / 每月2-3次 / 偶尔）
- 您最常去的品牌（零食很忙 / 好想来 / 其他）
- 您通常与谁一起购物（独自 / 与朋友 / 与家人）

---

### 模块二：价格与性价比感知（对应好评主题1）
> 数据挖掘发现：价格是最高频正面词，但也出现在差评"会员价格"中

- 您认为该店商品定价与超市相比：（便宜很多 / 略便宜 / 差不多 / 略贵）
- 您对会员折扣/优惠活动的满意度（1-5分）
- 您是否因为价格因素而放弃购买某商品？（是/否，如是请说明原因）
- 您认为该店性价比与竞争对手（零食量贩、便利店）相比如何？

---

### 模块三：产品品类与口味（对应好评主题2、3）
> 数据挖掘发现：口味和品类丰富是主要好评驱动

- 您最满意该店的产品品类（可多选）：饮料 / 坚果 / 巧克力/糖果 / 饼干/膨化 / 果冻/软糖 / 进口商品
- 您认为目前品类是否足够丰富？（非常丰富 / 基本满足 / 希望增加___类商品）
- 您对产品新鲜度/保质期管理的满意度（1-5分）
- **包装设计**：您是否遇到过零食包装难以打开的情况？（BERT新发现）（是/否，如是请描述）

---

### 模块四：门店体验（对应差评主题1、2）
> 数据挖掘发现：收银效率是最集中差评点

- 您在该店的平均等待结账时间（＜3分钟 / 3-8分钟 / 8-15分钟 / ＞15分钟）
- 对收银效率的满意度（1-5分）
- **请具体评价以下方面**（1-5分量表）：
  - 收银员服务态度
  - 员工主动推荐/引导能力
  - 货架整洁与补货及时性
  - 门店整体环境
- 您是否遇到过排队秩序混乱（如插队）的情况？（从未 / 偶尔 / 经常）

---

### 模块五：会员制度与数字化体验（对应差评主题3）
> 数据挖掘发现：会员兑换流程复杂是新兴痛点

- 您是否办理了该店会员？（是/否，如否原因是___）
- 您对会员积分/兑换规则的了解程度（完全了解 / 大致了解 / 不太了解 / 完全不了解）
- 对会员权益的整体满意度（1-5分）
- 您更希望通过什么方式了解会员优惠？（门店屏幕 / 微信小程序 / 店员介绍 / 货架标注）

---

### 模块六：开放性问题
> 用于捕获定量问卷无法覆盖的个性化反馈

- 您认为该店最需要改进的一点是什么？（开放填写）
- 您会向朋友推荐这家店吗？（NPS净推荐值：0-10分）
- 如果推荐/不推荐，主要原因是？

---

### 问卷设计要点总结

| 设计原则 | 说明 |
|---------|------|
| 聚焦高频痛点 | 收银效率单独设模块，细化等待时间和满意度 |
| 纳入新发现 | 包装开封问题（BERT独有发现）作为独立题目，验证普遍性 |
| NPS净推荐值 | 国际通用指标，便于与行业对比，适合正大杯量化分析 |
| 李克特量表 | 关键维度用1-5分，便于后续统计分析 |
| 人口统计前置 | 便于后续做分组分析（年轻人vs年长消费者的满意度差异） |
